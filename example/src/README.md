# Vibe-Model Example Repository

This is an example repository for testing the vibe-model tool's git integration features.

## Purpose

This repository serves as a test fixture for:
- Validating git operations
- Testing branch creation and merging
- Verifying commit message generation

## Structure

- `remote/` - Bare git repository (acts as the origin)
- `src/` - Working repository (clones/pushes to remote)
